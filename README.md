## Simple Controls
| Shortcut                       | Action                           |
|---------------------------------|----------------------------------|
| (ScrollWheel) or (Ctrl+/-)      | Zoom                             |
| (Click Window) or (Ctrl+W)     | Toggle cursor/window follow     |
